# Reference

Reference material for Garnish: the storage layout, environment variables, generated config files, the durable file schemas, and the dependency inventory. These pages document the concrete shapes and locations a contributor or integrator needs.

## Pages

- [Configuration](configuration.md) — Garnish storage layout, environment variables, and the generated config files (`config.yml`, `mcp.json`, the `garnish` shim).
- [Data models](data-models.md) — the durable file schemas: `events.jsonl`, `state.json`, `graph.json`, `quests.json`, `pack.yml`, and quest `.md` frontmatter.
- [Dependencies](dependencies.md) — npm dependencies and external integrations.

For the source-level building blocks, see [systems](../systems/index.md) and [primitives](../primitives/index.md). For how to build and test, see [tooling](../how-to-contribute/tooling.md).
